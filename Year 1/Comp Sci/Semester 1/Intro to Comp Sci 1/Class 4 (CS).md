## Switch Cases

```java
switch(condition)
{
	case 1:
		// do thing
		break;
	case 2:
		// do different thing
		break;
	default:
		// if the thing isnt there
		break;
}
```

## Ternary (shorthand if else)

```java
double x = 0.5;

x = if (x >= 0.5) ? 1 : x;
// if x is 0.5 or more, round it up to one, else just leave it


```
